# Sorting Algorithms in Python

Этот репозиторий содержит реализации популярных алгоритмов сортировки на языке **Python**.
Проект создан в учебных целях для изучения принципов работы алгоритмов сортировки.

## Реализованные алгоритмы

В проекте представлены следующие алгоритмы:

* **Bubble Sort** — сортировка пузырьком
* **Selection Sort** — сортировка выбором
* **Insertion Sort** — сортировка вставками
* **Merge Sort** — сортировка слиянием
* **Quick Sort** — быстрая сортировка

## Структура проекта


Bubble sort.py      # реализация Bubble Sort
Selection Sort.py   # реализация Selection Sort
Insertion sort.py   # реализация Insertion Sort
Merge sort.py       # реализация Merge Sort
Quick sort.py       # реализация Quick Sort


Каждый файл содержит отдельную реализацию алгоритма сортировки.

## Пример работы

Исходный массив:


[5, 2, 9, 1, 5, 6]


После сортировки:


[1, 2, 5, 5, 6, 9]


## Цель проекта

* Практика программирования на Python
* Изучение классических алгоритмов сортировки
* Понимание различий между алгоритмами по принципу работы
