# Sorting Algorithms in Python

Этот репозиторий содержит реализации популярных алгоритмов сортировки на языке **Python**.
Проект создан в учебных целях для изучения принципов работы алгоритмов сортировки.

## Реализованные алгоритмы

В проекте представлены следующие алгоритмы:

* **Bubble Sort** — сортировка пузырьком
* **Selection Sort** — сортировка выбором
* **Insertion Sort** — сортировка вставками
* **Merge Sort** — сортировка слиянием
* **Quick Sort** — быстрая сортировка
